# 1 目录结构
```shell
webapp(root-dir)
     |------WEB-INF
            |------classes(存放字节码)
            |------lib(第三方jar包)
            |------views(jsp页面)
            |------web.xml(注册Servlet配置文件)
     |------resources
            |------app
                    |------css
                    |------js
                    |------image
                    |------html
            |-------plugins
     README.md
     ...
```
